Indications sur l'évaluation des passes 1 et 2 du projet

Nous évaluons votre travail sur les points suivants:

  • Compilateur : votre compilateurs est évalué en faisant passer une base de tests. Notre base de tests comporte :
    • des tests incorrects (programmes JCas qui ne sont pas contextuellement correct). Nous vérifions que vous affichez un message d'erreur pertinent. Nous vérifions en particulier le numéro de ligne de la première erreur détectée.
    • des tests corrects. Nous vérifions que ces tests sont acceptés par votre compiltateur et que l'arbre abstrait décoré construit est correct.

  • Tests : votre base de tests est évaluée :
    • d'une part en examinant sa couverture sur notre compilateur de référence ;
    • d'autre part en calculant un score mutationnel. Pour cela, nous disposons d'environ 40 compilateurs comportant chacun un bug. Nous examinons si votre base de test permet de détecter chacun de ces bugs.
