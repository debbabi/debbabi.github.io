Passe 3 - Récupération du projet et compilation

  • Récupérer le projet depuis le dépot svn comme suit (remplacer votre_projet par le nom choisi pour votre projet, et bassem.debbabi@gmail.com par votre adresse Gmail):

    svn checkout https://votre_projet.googlecode.com/svn/trunk/ votre_projet --username bassem.debbabi@gmail.com
    Si le terminal vous demande un mot de passe, entrez celui que Google vous a généré ; celui-ci se trouve dans la page suivante sur Google code: https://code.google.com/hosting/settings

  • Dans le fichier .bash_profile, modifier le CLASSPATH : Remplacer $HOME par $HOME/votre_projet
  • Recompiler tout le projet :
    cd ~/votre_projet
    make # Compilation
    make html # Regénération de la documentation Html
