Installation de l'interprète de la machine abstraite (ima)

L'interprète de machine abstraite est un programme Ada. Vous devez donc avoir le compilateur Ada (commande gnatmake) pour l'utiliser.

  • Installer le compilateur Gnat (sur vos machines personnelles)
    Cliquer sur "Free Software or academic development"
    "Build your download package"
    Sélectionner votre plate-forme, puis sélectionner le fichier binaire de "GNAT_GPL"
    Terminer l'installation...

  • Installer la machine abstraite
    Téléchager les sources de la machine abstraite
    cd
    gunzip Interp_MA.tar.gz
    tar xvf Interp_MA.tar
    cd Interp_MA/Src
    make

    L'exécutable ima se trouve dans Interp_MA/Exec

    cd
    cp ~/Interp_MA/Exec/ima ~/ProjetCompil/Bin

  • Tester la machine abstraite
    Créer le fichier essai.ass contenant les lignes

    WSTR "ok"
WNL
HALT

    Effectuer la commande ima essai.ass
    Cette commande doit afficher "ok".
La documentation utilisateur de la machine abstraite se trouve dans ProjetCompil/Global/Doc/Ima.txt.
