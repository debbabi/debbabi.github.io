Vue d’ensemble

Le compilateur à développer, appelé JCas, comporte trois passes (le programme va être parcouru trois fois).

Introduction

  • Récupération et installation du projet

Passe 1

  • Analyse lexicale
  • Analyse syntaxique
  • Construction de l’arbre abstrait du programme

Passe 2

  • Vérifier qu’un programme JCas est contextuellement correct.
  • Décorer l’arbre abstrait du programme.

Passe 3

  • Parcourir l’arbre abstrait décoré une seconde fois et produire du code exécutable.
