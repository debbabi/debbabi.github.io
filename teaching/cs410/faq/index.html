TP Langages et Compilation


Foire aux questions

  • Utiliser Eclipse pour le dévelopement du projet
    Téléchargez le zip suivant, décompressez le sur votre environment de travail, déplacez votre dossier "ProjetCompil" sur le dossier "src" puis importez le projet "JCas" depuis eclipse.
    Pensez à mettre à jour le CLASSPATH pour prendre en compte ce nouveau chemin. La compilation (et les testes) se fait toujours depuis la console en utilisant "make".
  • BUG : remplacez le fichier "Environ.java" (qui se trouve dans "ProjetCompil/Global/Src") par ce fichier modifié. Il y avait une erreur dans la méthode afficher.
  • Problèmes liés à Cobertura : des solutions possibles..
    • rajouter l'option -XX:-UseSplitVerifier à la commande java dans le fichier testCouvertureVerif et à tous les fichiers appelés.(Merci à Cyril FISCHER).
    • changer JAVA_HOME pour qu'elle pointe vers la version 1.6 de Java. les étapes sont décrites sur ce fichier.
